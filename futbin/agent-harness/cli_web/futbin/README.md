# cli-web-futbin

> Generated by [CLI-Anything-Web](../../../../cli-anything-web-plugin/) from [futbin.com](https://www.futbin.com/)

Agent-native CLI for [FUTBIN](https://www.futbin.com/) — EA FC Ultimate Team player database.
Search players, check prices, compare stats, browse SBCs and Evolutions.

## Installation

```bash
cd futbin/agent-harness
pip install -e .
```

No authentication required — FUTBIN is a public site.

## Usage

### Player Search

```bash
cli-web-futbin players search --name "Mbappe" --json
cli-web-futbin players search --name "Salah" --evolutions
cli-web-futbin players search --name "Ronaldo" --year 25
```

### Player Detail

```bash
cli-web-futbin players get 40 --json
cli-web-futbin players get 21747 --json
```

### Player List (with filters + pagination)

```bash
# Cheapest gold strikers rated 85+
cli-web-futbin players list --position ST --rating-min 85 --cheapest --json

# Page 2 of results
cli-web-futbin players list --position ST --rating-min 85 --page 2 --json

# TOTY players under 500K
cli-web-futbin players list --version toty --max-price 500000 --json

# 5-star skill players
cli-web-futbin players list --min-skills 5 --json
```

**Filters:** `--position`, `--rating-min/max`, `--version`, `--min-price/max-price`, `--cheapest`, `--min-skills`, `--min-wf`, `--gender`, `--league`, `--nation`, `--club`, `--platform`, `--page`

### Player Comparison

```bash
cli-web-futbin players compare 40 21747 --json
```

Side-by-side stats with color-coded diffs in Rich table output.

### Market Index

```bash
cli-web-futbin market index --json
```

### Squad Building Challenges

```bash
cli-web-futbin sbc list --json
cli-web-futbin sbc list --category Players --json
cli-web-futbin sbc get 665 --json
```

### Evolutions

```bash
cli-web-futbin evolutions list --json
cli-web-futbin evolutions list --expiring --json
cli-web-futbin evolutions get 666 --json
```

### Config (Persistent Defaults)

```bash
cli-web-futbin config set year 25        # Set default year
cli-web-futbin config set platform pc    # Set default platform
cli-web-futbin config show --json        # Show current config
cli-web-futbin config reset              # Reset to defaults (year=26, platform=ps)
```

Once set, `--year` and `--platform` are optional on all commands.

### Auth (Optional)

FUTBIN is a public site — no auth required.

```bash
cli-web-futbin auth status --json
```

### REPL Mode

```bash
cli-web-futbin
# Enters interactive REPL with sectioned help
```

## JSON Output

Every command supports `--json`. Errors also return structured JSON:

```json
{"error": true, "code": "NOT_FOUND", "message": "Player 999999 not found"}
```

Error codes: `NOT_FOUND`, `NETWORK_ERROR`, `RATE_LIMITED`, `PARSING_ERROR`, `SERVER_ERROR`, `INVALID_INPUT`

## Notes

- Prices are in EA FC coins
- Year 26 = EA FC 26 (current), 25 = EA FC 25
- Rate limiting: 0.5s delay between requests
- Config stored at `~/.config/cli-web-futbin/config.json`
